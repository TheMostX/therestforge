## 🚀 What is restforge?

`restforge` parses OpenAPI 3.x specs and generates production-ready, fully typed API clients for JavaScript/TypeScript, Python, and Go — with built-in error handling, retry logic, and request/response types.

```bash
restforge generate openapi.json --lang typescript --out ./src/api
restforge generate openapi.yaml --lang python --out ./lib/client
restforge generate openapi.json --lang go --package myclient --out ./client
restforge preview openapi.json          # Preview what will be generated
restforge validate openapi.json         # Validate the spec first
restforge demo                          # Generate from a sample spec
```

## ✨ Features
- 📄 OpenAPI 3.x and Swagger 2.x support
- 🟦 TypeScript — typed interfaces + fetch client
- 🐍 Python — dataclasses + httpx client
- 🐹 Go — structs + net/http client
- 🔁 Auto-retry with exponential backoff
- 🔐 Auth helpers (Bearer, API key, OAuth2)
- 📋 JSDoc / docstring generation from descriptions

## 📄 Sample Generated TypeScript
```typescript
// Auto-generated by restforge — DO NOT EDIT
export interface User { id: number; name: string; email: string; }

export async function getUser(id: number): Promise<User> {
  const res = await fetch(`/api/users/${id}`, {
    headers: { Authorization: `Bearer ${API_KEY}` }
  });
  if (!res.ok) throw new APIError(res.status, await res.text());
  return res.json() as Promise<User>;
}
```
